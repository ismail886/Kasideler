# Kasideler

Bu Android uygulaması, çeşitli kasideleri listeleyen ve dinlemenize olanak tanıyan modern bir Jetpack Compose uygulamasıdır.

## Özellikler

*   **Kaside Listesi:** Farklı yazarların ve hocaların seslendirdiği kasideleri görüntüleyin.
*   **Detay Görünümü:** Kasidelerin özetlerini ve tam metinlerini okuyun.
*   **Ses Oynatıcı:** Entegre MediaPlayer ile kasideleri uygulama içinden dinleyin.
*   **Modern Tasarım:** Material 3 ve Jetpack Compose kullanılarak geliştirilmiş şık ve kullanıcı dostu arayüz.
*   **Dinamik Renkler:** Her kaside için özel gradyan renk temaları.

## Kullanılan Teknolojiler

*   **Kotlin:** Programlama dili.
*   **Jetpack Compose:** Modern Android UI kiti.
*   **Material 3:** Modern tasarım bileşenleri.
*   **MediaPlayer:** Ses dosyalarını oynatmak için.

## Kurulum

1.  Bu depoyu klonlayın:
    ```bash
    git clone https://github.com/kullaniciadi/Kasideler.git
    ```
2.  Android Studio'yu açın ve projeyi içe aktarın.
3.  Projeyi bir emülatörde veya gerçek bir cihazda çalıştırın.
